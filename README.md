# VidiMatch - Videographer Marketplace

A React-based marketplace connecting clients with professional videographers.

## Features
- Post videography projects
- Browse open projects
- Real-time updates
- Responsive design

## Setup
1. Clone repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure Firebase values
4. Run `npm start`

## Tech Stack
- React
- Firebase (Firestore, Auth)
- Tailwind CSS
- Lucide React icons
